# Mini-AHU · Сравнение приточных установок

Онлайн-сервис сравнения компактных приточных установок (Mini-AHU) в минималистичном стиле страницы сравнения Apple.

**Демо:** включите GitHub Pages (Settings → Pages → Branch: `main`, папка `/root`) и откройте выданный адрес.

## Возможности
- Поиск по названию/бренду, фильтры (бренд, расход воздуха, цена, наличие клапана/фильтра).
- Выбор до **4 моделей** для сравнения, панель выбранного, удаление.
- Приоритетные параметры сверху и крупно: **расход воздуха, уровень шума, класс фильтрации**.
- Остальные характеристики — секциями-аккордеонами (цена, габариты + выделенная толщина, питание, клапан, рекуперация, нагреватель, охладитель, вентилятор, управление, IP).
- Прилипающая шапка с названиями моделей, подсветка различий, адаптив (мобайл → десктоп), горизонтальный скролл, поделиться ссылкой (состояние в URL `?c=`).

## Данные
- `data.json` — 232 приточные установки (VentMachine, Breezart, Horynize, Minibox, Turkov, Rusklimat/SHUFT), сгенерирован из листа «Данные» файла `Конкурентный_анализ.xlsx`. Приточно-вытяжные (ПВУ) исключены. Полный приточный модельный ряд Breezart (до 6000 м³/ч) собран с breezart.shop (OFFERS + прайс-лист 2026): цены, габариты, характеристики.
- Источник — открытые сайты производителей (сбор 2026-05-29). «н/д» = параметр не опубликован. Цены справочные.

## Технологии
Чистые HTML + CSS + JavaScript, без зависимостей и сборки. Хостинг — любой статический (GitHub Pages).

## Локальный запуск
```bash
python -m http.server 8000
# открыть http://localhost:8000
```
